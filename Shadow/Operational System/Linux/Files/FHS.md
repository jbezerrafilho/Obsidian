#FHS 

### FileSystem Hierachy Standard

```shell
/     - Root do Sistema
/boot - Arquivos de boot
/etc  - Files setup system, packages network default
/bin  - Binários programas, comandos básicos dos users sistema
/sbin - Binários do Root para administração do sistema\
/var  - Contém os logs do sistema, spool de impressora e cache
/root - Home do root
/home - Home dos usuários
/dev  - Acesso aos devices do sistema
/lib  - Bibliotecas do sistema
/proc - Sistema arquivos do Kernel. Não existe no HD.
/usr  - Contém arquivos e apps dos usuários do sistema

```
